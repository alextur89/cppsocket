CppSocket
=============================

CppSocket project provides the common socket functionality

Platform
==========

* Linux
* Mac Os

### Content:
    tests/ - unit tests via google test;
    source/ - source code;
    include/ - headers;
    doc/ - docs generated by doxygen;
    CMakeLists.txt - cmake conf file;
    README.md - this file;

Build
======
~~~
cmake .
make
make test (for testing)
~~~
